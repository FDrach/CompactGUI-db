# CompactGUI Game Database Front-End

 [fdrach.github.io/CompactGUI-db/](https://fdrach.github.io/CompactGUI-db/).

Simple app that lets you search, sort, and switch between multiple layouts to compare CompactGUI compression outcomes sourced from the community database.
